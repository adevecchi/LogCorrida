- Descrição
  
  Este projeto está escrito em JavaScript no lado do cliente e tem uma interface mínima em HTML para o carregamento do arquivo de log
  e a apresentação do resultado final do processamento do arquivo de log com a classificação dos pilotos da corrida.

- Instruções
  
  Depois de clonar ou baixar o arquivo compactado, o mesmo pode ser testato com um duplo clique no arquivo index.html sendo aberto
  o navegador padrão como arquivo local ou pode-se executar em um servidor web como Apache, IIS da Microsoft ou outro de sua escolha.
  
  Uma alternativa é utilizar o servidor web embutido do PHP, para isso entre no diretório dos arquivos e abra o prompt do DOS para
  Windows ou o terminal do Linux e digite o seguinte comando:
  
  Windows (supondo que o diretório esteja em C:\LogCorriga)
  
    C:\LogCorrida>php -S localhost:8000
    
  Linux (supondo que o diretório esteja em /home/nome_usuario/LogCorrida)
    
    $ php -S localhost:8000
  
  Para o caso de estar sendo usado o servidor web embutido do PHP e após a execução do comando acima,
  abrir o navegador web e digitar a url na barra de endereço: localhost:8000
  
  Com o navegado web aberto, carregar o arquivo corrida.log que esta presente no diretório e contem os dados de log da corrida
  para ser apresentado qual foi a classificação final da corrida dos pilotos.
